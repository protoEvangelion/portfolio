To setup **Typescript validation,** simply set it to true in VSCode settings:

```plain
"typescript.validate.enable": true,
```

I had turned this off in my workspace settings in the past and forgot! So make sure to double check that this setting is not set to false in your **user settings, workspace settings or folder settings**.
